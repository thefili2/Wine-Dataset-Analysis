This project performs an exploratory data analysis on a wine dataset to investigate the relationship between physicochemical properties and wine quality. The dataset is split into two groups based on whether the quality score is above or below the global average. For each group, statistical summaries (mean values) are computed across all chemical features, including acidity, sugar content, sulfur dioxide levels, density, pH, sulphates, and alcohol. The results are then visualized using a comparative horizontal bar chart to highlight structural differences between the two groups. The analysis provides an initial understanding of which variables may be associated with higher quality wines and serves as a foundation for further statistical modeling or machine learning approaches.
